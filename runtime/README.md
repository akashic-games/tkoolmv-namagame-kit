# tkoolmv-namagame-runtime

ツクール MV で作成したゲームをニコ生ゲームとして動作させるためのライブラリです。

## ビルド方法

```sh
npm install
```

## 非サポート要件
- タイトルシーンのメニュー表示
  - セーブ・ロードに非対応のため、メニューは表示されません。一定時間で自動的にニューゲームが選択されます。
- 30MB 以上の容量
  - ニコ生ゲームのサイズ上限のため ([詳細][size-limit])
- 音声のシーク再生
- 音声のピッチ・位相の変更
- セーブ・ロード機能
- オプション機能
- 動画再生
- Window.png によるウィンドウデザインの一部カスタマイズ
- キーボード入力
- フォントの変更 (ブラウザ依存となります)
